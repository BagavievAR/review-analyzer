# Web App for Review Analysis

Учебный проект: веб-приложение на Python/Flask для сбора и анализа пользовательских отзывов с использованием методов обработки естественного языка.

## Tech Stack

- Python
- Flask
- spaCy
- SQLite
- Docker

## Features

- Сбор пользовательских отзывов через веб-интерфейс
- Хранение отзывов в базе данных (SQLite)
- Анализ тональности и ключевых слов с помощью spaCy
- Простая админ-панель для просмотра и фильтрации отзывов
- Контейнеризация приложения с использованием Docker

## How to Run (local, via venv)

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/<your-username>/review-analyzer.git
   cd review-analyzer
   ```

2. Создать и активировать виртуальное окружение:
   ```bash
   python -m venv venv

   # Windows (PowerShell)
   .\venv\Scripts\Activate.ps1

   # или Windows (cmd)
   venv\Scripts\activate.bat
   ```

3. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Инициализировать базу данных:
   ```bash
   flask --app run init-db
   ```

5. Запустить приложение:
   ```bash
   python run.py
   ```

6. Открыть в браузере:
   - http://127.0.0.1:5000/
